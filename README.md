Senior Backend Engineer with 10+ years building scalable SaaS platforms and high-performance backend systems.

Core stack: Python, Django/DRF, FastAPI, Redis, and distributed backend architecture.

I design secure, data-driven APIs, background processing pipelines, and caching layers focused on reliability, scalability, and maintainability.

I led an optimization that replaced part of internal latency-sensitive HTTP communication with RabbitMQ/AMQP messaging between services, while keeping serializer-based validation standards. This reduced network usage, improved responsiveness, and lowered infrastructure costs.

I have delivered software across EdTech, retail, accounting, and business intelligence, and contribute through system design, code reviews, and cross-team collaboration.

I also bring practical Web3 experience (wallets, DeFi integrations, and Layer 2 networks such as Polygon, Optimism, and Arbitrum) when hybrid Web2 + Web3 architecture creates clear product value.

Keywords: Senior Backend Engineer, Python, Django, DRF, FastAPI, REST APIs, Microservices, Distributed Systems, RabbitMQ, AMQP, Redis, PostgreSQL, Docker, SaaS, System Design, Performance Optimization, Scalability, Cloud-ready Architecture.
